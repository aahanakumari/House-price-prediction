# House-price-prediction
This project is a simple machine learning model built using Python to solve a real-world prediction problem. It involves data preprocessing, exploratory data analysis, feature engineering, and model training using scikit-learn. The goal  is to understand the machine learning workflow and apply it to a structured dataset to generate  predictions.  
## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Steps
- Data cleaning
- Feature engineering
- Model training
- Evaluation

## Result
Achieved good accuracy using regression models.
